# StoryFlow

## Project Overview
This project aims to create an interactive storytelling platform that utilizes AI to assist users in crafting engaging narratives.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vasireddy/StoryFlow.git
   cd StoryFlow
   ```
2. Create a `.env` file in the root directory and add your environment variables there:
   ```bash
   # Example variable
   API_KEY=your_api_key_here
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   node src/app.js
   ```
